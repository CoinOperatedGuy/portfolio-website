Intro

Hi, my name is Trevor Jefferson and i'm part-time student at Lambda School. I'm currently studying to be a full-stack web developer. I'm still in the learning process but so far i think i'm doing pretty good.

I started coding a few months and at first, it was out of curiosity. Soon enough, it felt like a coding bug had bit me because I then dedicated almost all my free time to coding. There would be times where I would be like "Ok, it's 11PM. I'm gonna get off and get some rest.". Next thing I know, it's 3AM and I have to get up in a couple hours. Then I discovered Career Karma. With their help, I was able to research and apply to several bootcamps, including the one I'm in now. When I got accepted, I was so stoked that I might have scared some people when I burst into cheer.

So now that I've told you a little about myself, please feel free to explore the rest of this site. By the way, check out my awesome work.
